# Configuracion inicial
En este archivo se encuentra la configuracion inicial del linter **eslint** y del format **prettier**.

## paquetes instalados
- vite
- eslint (standar)
- prettier
- eslint-config-prettier
- react

### clonar repositorio
`git clone https://github.com/xavii07/config-start-vite.git`

### instalar paquetes
`npm install`

### iniciar proyecto
`npm run dev`
